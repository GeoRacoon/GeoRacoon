# Landiv example scripts

This is a collection of scripts that apply the landiv package.

They can be considered as usage examples, but they might even implement a complete analysis.


_Note:_ 

For these scripts to run fine, you should coreate a `locals.py` script in which you set the
variables `data_path` and `results_path` according to your system.
[locals.py.example](./locals.py.example) is an example of how this file could look like.
